There is something wrong with the page link. Before looking at the solution please try to look at the syntax for the .js file and make sure everything is correct. If everything is correct the .js file should alert a message on the screen.

Note: One can check the console area to see the error. The error should look something like this: 404 (Not Found)

  • To access the console area in Google Chrome right click screen(not on any text) and then choose "Inspect Element". Then click "console."
  • To access the console area in Firefox install the Firebug plugin. There should be documentation with the plugin.
